# UNIT3D - Telegram Webhook Issue & Fix Plan

## Resumen Ejecutivo

**Problema:** El recuadro de vinculación Telegram de RawSmoke muestra "vincular con el bot" aunque se generó un token válido el 2026-03-27 11:46. Ayer se regeneró el token pero nunca completó el handshake.

**Causa Raíz Identificada:** El webhook de Telegram nunca fue registrado. La API de Telegram devuelve:
```json
{"url": "", "pending_update_count": 0}
```
Sin URL registrada, Telegram nunca envía los mensajes `/start` del usuario al endpoint `/api/telegram/webhook`.

**No es un bug de código** — el controlador `TelegramWebhookController` está implementado correctamente. Es un problema de **setup/configuración**.
